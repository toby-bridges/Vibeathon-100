## 项目简介

如今 AI 的发展，让我想起了大学时看《疑犯追踪》时的爽感和后背发凉。这个项目正是对它的致敬。

The Machine —— 一个灵感来自《疑犯追踪》(Person of Interest) 的人工智能系统交互页面。正如剧中那句经典的使命宣言：**Predict. Prevent. Protect.**（预测、预防、保护）

## 核心特点

- 沉浸式 AI 系统界面设计
- 深色/浅色主题自适应
- 响应式布局
- 向经典美剧致敬的交互体验

## 技术栈

- Next.js
- React
- Tailwind CSS
- Geist 设计系统

## 收获与反思

第五天的挑战，把曾经追剧时的幻想变成现实。AI 时代已来，The Machine 不再只是科幻。
