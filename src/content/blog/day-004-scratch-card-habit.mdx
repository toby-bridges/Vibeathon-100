## Project Introduction

Scratch Card Habit Tracker is a gamified habit management app. It addresses the pain point of users often forgetting to check in or finding check-ins burdensome. By replacing traditional habit tracking with scratch card interactions, habits are hidden within scratch cards, reducing pressure while adding surprise and engagement.

## Core Features

- Gamified Interface: Scratch card mechanics replace traditional habit tracking
- Habit Management: Create and manage multiple habits
- Data Statistics: Track progress analytics
- Interactive Design: Prototype-based user engagement

## Tech Stack

- TypeScript
- Next.js
- Tailwind CSS
- Prisma ORM
- Vercel Deployment
