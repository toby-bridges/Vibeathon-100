## 项目简介

NomadKit 是一款数字游民物品标注工具，帮助用户快速标注物品照片。通过 AI 自动识别物品位置，生成带气泡标注的图片，便于在社交平台分享。

## 核心功能

- **AI 物品识别**：基于 Gemini Vision API 自动识别图片中的物品
- **三种场景模式**：EDC（随身物品）、Setup（桌面工作站）、Furniture（家具家电）
- **可视化标注编辑**：拖拽调整气泡位置，编辑物品名称和价格
- **一键导出**：生成并导出带标注的图片

## 技术栈

- Next.js 16.1.1 - 全栈框架
- React 19.2.3 - UI 组件库
- TypeScript 5.x - 类型安全
- Tailwind CSS 4.x - 样式框架
- Google Gemini API - AI 视觉识别
- Canvas API - 图片处理
