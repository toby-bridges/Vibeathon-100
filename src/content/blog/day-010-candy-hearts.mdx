## 项目简介

告别仪式（Candy Hearts）是一款帮助年轻女性以温柔方式处理前任照片的仪式感应用，灵感源自《老友记》中的净化仪式概念。通过三种不同的告别方式，让分手后的情感处理变得更有仪式感和治愈感。

## 核心功能

- **封存（上滑）**：模糊封印照片，可设定未来开启日期
- **抹除（左滑）**：通过粒子效果让照片消散
- **珍藏（右滑）**：应用柔光滤镜并记录想说的话
- **多种视觉效果**：马赛克、蒲公英粒子、灰飞烟灭等特效

## 技术栈

- Next.js 14 (App Router) - 全栈框架
- TypeScript - 类型安全
- Tailwind CSS + PostCSS - 样式框架
- Framer Motion - 动画效果
- Canvas API - 图像处理
- heic2any - HEIC 格式兼容
