## 项目简介

当"痛点"这个词被滥用时，我们没有意识到这个世界上真的有人和韭二一样饱受慢性疼痛困扰。

这款应用帮助用户用精确的医学术语记录疼痛，让与医生的沟通更加高效。

## 核心功能

- **交互式人体图谱**：点击身体部位即可记录疼痛位置
- **正面/背面视图**：全方位标记疼痛区域
- **历史记录**：追踪过往疼痛数据
- **报告生成**：将记录转化为医学报告

## 技术栈

- Next.js
- React
- Tailwind CSS
- PWA（支持离线使用）
